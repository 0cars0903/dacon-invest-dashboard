# Skills.md 모듈 설계 — InvestLens

> 4개 모듈로 분리하여 각 역할을 명확히 정의.
> 제출 시 `.zip`으로 묶어 제출.

## 모듈 구성

### 1. 데이터감지규칙.md
**역할**: 업로드된 데이터의 유형을 자동 판별
- 컬럼명 패턴 매칭 (Date, Close, Volume → 주가 시계열)
- 값 범위/타입 검사 (0~1 → 비중, 양수+음수 → 수익률)
- 데이터 유형 분류: `stock_timeseries`, `etf_comparison`, `portfolio_allocation`, `custom`
- fallback 규칙: 패턴 미매칭 시 기본 시계열 처리
- **[v2.2]** 데이터 신뢰성 검증 3규칙 추가 (날짜 연속성, 가격 이상치, 최신성)

### 2. 지표계산규칙.md
**역할**: 감지된 유형에 따라 자동 계산할 투자 지표 정의
- **주가 시계열**: 일간수익률, 누적수익률, 이동평균(20/60/120), RSI, 변동성(20일)
- **ETF 비교**: 기간수익률, 샤프비율, 최대낙폭(MDD), 상관계수 행렬
- **포트폴리오**: 가중수익률, 자산배분비율, 리밸런싱 시뮬레이션
- 각 지표에 대한 계산 공식, 입력/출력 스키마 명시
- **[v2.2]** ETF 전용 지표 3종 추가: 괴리율(Premium/Discount), NAV 대비 위치, 추적오차(Tracking Error)

### 3. 시각화매핑규칙.md
**역할**: 데이터 특성에 맞는 차트 타입 자동 선택
- 시계열 데이터 → LineChart (가격 추이), AreaChart (누적 수익률)
- 비교 데이터 → BarChart (기간 수익률 비교), RadarChart (지표 비교)
- 구성 데이터 → PieChart (자산 배분), TreeMap (섹터별 비중)
- 분포 데이터 → Histogram (수익률 분포), ScatterPlot (위험-수익)
- 차트 우선순위, 대시보드 배치 규칙, 색상 팔레트 기준
- **[v2.2]** 지표별 경고 색상 임계값 테이블 추가 (Tailwind 클래스 기준)
- **[v2.2]** Level 1~3 점진적 공개 전환 기준 구체화 (전환 트리거, 차트별 Level 매핑)

### 4. 인사이트생성규칙.md
**역할**: 수치 분석 결과를 사람이 읽을 수 있는 인사이트로 변환
- 조건부 해석 템플릿: "수익률 > 0 → '상승 추세', < -5% → '급락 경고'"
- 비교 인사이트: "A가 B 대비 X% 아웃퍼폼"
- 요약 카드 문구 생성 규칙
- 심각도 레벨: info / warning / alert
- 인사이트 우선순위 (가장 중요한 것 상단 배치)
- **[v2.2]** 초보자 친화 메시지 변환 템플릿 5종 추가 (괴리율, MDD, 수익률, 크로스오버, 포트폴리오 성향)
