# 🎯 서비스 컨셉 — InvestLens

> "어떤 투자 데이터든 업로드하면, Skills.md가 자동으로 분석하고 대시보드를 만든다."

---

## 한 줄 정의
**InvestLens** — Skills.md 기반 범용 투자 데이터 자동 분석 대시보드

---

## 핵심 가치
사용자가 주식, ETF, 포트폴리오 등 어떤 형태의 투자 데이터(CSV/JSON)를 올려도,
Skills.md에 정의된 **분석 규칙**에 따라 자동으로 데이터를 감지 → 지표 계산 → 시각화 → 인사이트 생성.

"데이터 구조가 매번 다르고 분석 기준도 달라" 일관된 시각화가 어려운 문제를 Skills.md로 해결한다.

---

## 작동 흐름

```
[사용자] CSV/JSON 업로드
    │
    ▼
[Phase 1: 데이터 감지]
    Skills.md 규칙으로 데이터 유형 자동 판별
    → 주가 시계열? ETF 비교? 포트폴리오 구성?
    │
    ▼
[Phase 2: 지표 계산]
    Skills.md 정의에 따라 적절한 투자 지표 자동 계산
    → 수익률, 변동성, 샤프비율, 상관관계 등
    │
    ▼
[Phase 3: 시각화 매핑]
    Skills.md 차트 선택 기준에 따라 자동 매칭
    → 시계열 → 라인차트, 비교 → 바차트, 분포 → 히스토그램 등
    │
    ▼
[Phase 4: 인사이트 생성]
    Skills.md 규칙에 따라 자동 해석 문구 생성
    → "최근 30일 수익률이 -5.2%로 하락 추세입니다"
    │
    ▼
[대시보드 렌더링]
    위 결과를 자동 레이아웃으로 구성
```

---

## 평가 항목별 공략

### 범용성 (25점) — "어떤 투자 데이터든"
- CSV/JSON 업로드 기반이므로 데이터 소스에 구애받지 않음
- 데이터 유형 자동 감지: 컬럼 이름 패턴, 값 범위, 시계열 여부로 판단
- 데모 데이터셋 3종 내장: 주가 시계열, ETF 비교, 포트폴리오 구성

### Skills.md 설계 (25점) — "분석 규칙의 명확성"
Skills.md를 4개 모듈로 분리:

| 모듈 | 역할 |
|------|------|
| `데이터감지규칙.md` | 컬럼 패턴 매칭, 데이터 유형 분류 기준 |
| `지표계산규칙.md` | 유형별 계산할 투자 지표 정의 |
| `시각화매핑규칙.md` | 데이터 특성 → 차트 타입 선택 기준 |
| `인사이트생성규칙.md` | 수치 해석 규칙, 문구 생성 템플릿 |

### 대시보드 자동 생성 (25점) — "업로드만 하면 끝"
- 파일 업로드 → 자동 분석 → 대시보드 렌더링까지 원클릭
- Recharts 기반 반응형 차트
- 카드 레이아웃 자동 구성

### 바이브코딩 활용 (15점) — "Claude Cowork 전면 활용"
- Skills.md 자체가 Claude에게 주는 규칙 문서
- 프로젝트 전체를 Cowork 바이브코딩으로 구현
- CLAUDE.md에 작업 지침 정의 → Skills.md 참조 구조

### 실용성 및 창의성 (10점) — "실제 쓸 수 있는 도구"
- 데모 데이터로 즉시 체험 가능
- 다크모드 지원
- 분석 결과 PDF 내보내기 (확장 기능)

## 데모 데이터셋 (3종 내장)

| # | 이름 | 설명 | 컬럼 예시 |
|---|------|------|-----------|
| 1 | 주가 시계열 | AAPL, MSFT 등 5종목 1년 일봉 | Date, Open, High, Low, Close, Volume |
